# 08. CSS

## Archivo

1. `static/css/site.css`

## Objetivo

1. Estilos globales y componentes.
2. Tema base para pacientes.
3. Tema especial para doctor (colores y tipografia distinta).

## Componentes clave

1. `.topbar`, `.navbar`, `.footer`.
2. `.hero`, `.card`, `.feature`, `.list-item`.
3. `.form` para formularios.
4. `.message` para mensajes.

## Tema Doctor

1. Clase `role-doctor` ajusta colores, contrastes y botones.
2. Formularios del doctor con fondo blanco y texto oscuro.
3. Elementos de recomendaciones y cajas informativas.
