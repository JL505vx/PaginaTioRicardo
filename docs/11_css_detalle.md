# 11. CSS en Detalle

Este documento describe clases y componentes principales.

## Layout y estructura

1. `.site`: layout principal.
2. `.container`: ancho y centrado.
3. `.topbar`, `.navbar`, `.footer`.

## Botones

1. `.btn` base.
2. `.btn--primary`, `.btn--dark`, `.btn--ghost`.

## Cards y listas

1. `.card`: tarjetas.
2. `.feature`: tarjetas de beneficios.
3. `.list` y `.list-item`.
4. `.pill`: indicador.

## Formularios

1. `.form`.
2. Inputs y textarea.

## Mensajes y chat

1. `.message`, `.message__meta`.

## Tema Doctor

1. `.role-doctor`: tema especial.
2. `.doctor-shell`: ajustes en panel doctor.
3. Inputs del doctor con fondo blanco.
